# l2g
A webapp to convert given local schemas to a single global schema

## Steps to install
- `cd l2g`
- `chmod +x install.sh`
- `./install.sh`

## Steps to run
Open 2 terminals. One is for the frontend and other is for the backend

### Frontend server
- `cd l2g`
- `cd frontend`
- `npm start`

### Backend server
- `cd l2g`
- `cd backend`
- `flask run`
